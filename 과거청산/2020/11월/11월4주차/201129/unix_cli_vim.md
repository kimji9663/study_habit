#UNIX CLI

## 텍스트에디터 : VIM

CLI 환경에서 사용하는 텍스트 에디터.
vi 라는 텍스트 에디터 향상 버전.

* Vim 공식 사용 설명서(https://vimhelp.org/#help.txt)
* Vim을 게임처럼 재미있게 배울 수 있는 사이트(https://vim-adventures.com/)

### VIM 실행
```
vim            //새 파일 만들기
vim [파일명]
```

 ### 모드전환
 
esc : 각 모드에서 일반모드로 전환<br>
왼쪽 아래에 현재 모드가 나타남.

 1. 입력 모드
  - i 
  - a : 커서를 하나 뒤로 보내고 입력모드로 전환 
  - I : 커서를 가장 앞으로 보내고 입력모드로 전환
  - A : 커서를 가장 뒤로 보내고 입력모드로 전환
  - o : 커서를 다음 줄로 옮기고 입력모드로 전환
  - O : 윗줄에 빈 줄이 생김, 커서를 빈 줄로 옮기고 입력모드로 전환

2. 비주얼 모드
 - v : 한 칸씩 블록 지정 할 때
 - V : 줄 단위로 블록 지정 할 떼

3. 명령모드 
 - : : 저장, 종료 할 때
 - / : 텍스트 검색 및 치환할 때


### 1.일반 모드
vim 실행 후 첫 화면
 1. 커서이동
* 방향키
* 알파벳 h(왼쪽), j(아래쪽), k(위쪽), i(오른쪽)을 사용하여 이동할 수 있다.
ex) 오른쪽으로 5칸 이동 => 5누른 후 i키
* ctrl + g : 현재 커서 위치를 알 수 있다.
* 0 : 커서가 줄의 맨 앞으로 이동
* $ : 커서가 줄의 맨 뒤로 이동
* gg : 첫번재줄의 맨 앞으로 이동
* G : 마지막 줄의 맨 앞으로 이동

 2. 텍스트 삭제
 * x : 텍스트를 한 칸씩 삭제
 * 8x : 텍스트를 8칸 삭제
 * dd : 문장 삭제
 * 2dd : 문장 2줄 삭제

 3. u : 이전 작업 취소

 4. 붙여넣기
 p : 커서 다음 칸에 붙여넣기
 P : 커서 이전 칸에 붙여넣기

### 2.입력 모드
 - 텍스트 입력

### 3.비주얼 모드 
 - 방향키 : 텍스트 블록 지정
 - x : 텍스트 블록 삭제
 - y : 텍스트 블록 복사
 - d : 텍스트 잘라내기

### 4.명령 모드
1. :
 - w : 내용 저장
 ```
 :W           // 기존 파일을 수정하는 경우
 :w [파일명]  // 새 파일 저장하는 경우
 ```  
 - wq : 저장하고 vim 종료
 - q : vim 종료
 - q! : 저장하지 않고 vim 종료

 2. / : 특정 텍스트 검색 및 치환
 ```
 /[단어명]  // 특정 텍스트 검색
 ```
- n : 다음 검색 내용으로 이동
- N : 이전 검색 내용으로 이동
- s : 텍스트 대체
```
:s/[단어명]/[대체할단어명]    // 한 개씩 텍스트 변경
:%s/[단어명]/[대체할단어명]   //각 줄의 첫번째 텍스트 변경
:%s/[단어명]/[대체할단어명]/g //전체 텍스트 변경
:%s/[단어명]/[대체할단어명]/gc //전체 텍스트를 확인하면서 변경
```
