조건: 반복이 시작되기 전에 조건문은 참, 거짓을 판단하게 된다. 만약 조건문이 참이라면, while문 안의 문장들이 실행된다. 거짓이라면 문장은 그냥 while 반복문 후로 넘어간다. var n = 0; var x = 0; while (n < 3) { n++; x += n; } * 다음의 while문은 n이 3보다 작을 때까지 반복한다. 반복을 살펴보면, n을 x에 계속 더하게 된다. 그러므로 x와 n변수는 다음의 값을 갖는다. - 첫번째 반복; n=1 과 x=1 - 두번째 반복; n=2 과 x=3 - 세번째 반복; n=3 과 x=6 세번째 반복후, n< 3이라는 조건은 더 이상 참이 아니라 되므로 반복은 종료된다.
